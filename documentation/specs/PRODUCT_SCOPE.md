# Product Scope

## Product Name
AI Career Agent

## Version
V1

## Purpose
Help the user discover and prioritize relevant jobs.

## User Outcome
The user opens the dashboard and sees:
- newly collected jobs
- match score
- why each job matches
- manual tracking status

## Must Have
- local execution
- Docker support
- SQLite
- rules-based scoring
- Streamlit dashboard
- tests
- final implementation report

## Nice to Have
- mock/demo jobs
- source health logging
- import/export CSV

## Must Not Have
- automatic CV submission
- auto apply workflows
- browser automation for applications
- CAPTCHA bypass
- hidden background behavior

## Manual Status Values
- new
- reviewing
- saved
- ignored
- applied_manual
