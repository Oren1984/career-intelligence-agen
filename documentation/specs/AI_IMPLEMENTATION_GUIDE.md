# Claude Agent Instructions

You are responsible for implementing the AI Career Agent system.

Your objectives:

1. Implement a modular Python codebase.
2. Create job collectors.
3. Implement filtering and scoring.
4. Store jobs in SQLite.
5. Provide a Streamlit dashboard.
6. Ensure the system runs inside Docker.

---

# Development Phases

Phase 1
Create project skeleton and database schema.

Phase 2
Implement job collectors.

Phase 3
Implement filtering logic.

Phase 4
Implement match scoring.

Phase 5
Create Streamlit dashboard.

Phase 6
Add logging and reports.

---

# Constraints

Do NOT implement:

- automatic job applications
- browser automation
- CAPTCHA solving

The system must remain a decision-support assistant.

---

# Deliverables

Code modules
Tests
Documentation
Final report summarizing:

- number of jobs collected
- scoring results
- system health
