# Definition of Done

Version 1 is complete only if all items below are true:

1. The repository contains a working modular codebase
2. SQLite database initializes successfully
3. At least one collector works
4. Mock/demo data path exists for stable demonstration
5. Jobs are normalized and deduplicated
6. Rules-based filtering works
7. Scoring engine works
8. Streamlit dashboard loads and shows jobs
9. Manual status updates work
10. Docker setup runs successfully
11. Tests exist and run with pytest
12. FINAL_IMPLEMENTATION_REPORT.md exists
13. TEST_RESULTS.md exists
14. The implementation does not auto-apply to jobs
