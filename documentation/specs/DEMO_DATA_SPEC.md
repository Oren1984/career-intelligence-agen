# Demo Data Specification

The repository should include a small demo dataset for local testing and dashboard demonstration.

## Required Demo Jobs
At least 10 example jobs:
- Applied AI Engineer
- MLOps Engineer
- AI Engineer
- Data Engineer
- Python Backend Engineer
- Observability Engineer

## Variation
Include:
- high match examples
- medium match examples
- low match examples
- negative examples with senior/phd/relo flags

## Purpose
This ensures the dashboard, scoring, and tests can run even without live scraping.
