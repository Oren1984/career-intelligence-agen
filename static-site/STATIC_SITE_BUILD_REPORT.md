A high-end static portfolio website presenting the **Career Intelligence Agent** system.

Built for:

* 🎯 Portfolio showcase

## 🚀 Overview

This site is a **static presentation layer** of the Career Intelligence Agent — a privacy-first, local decision-support system for career intelligence.

> ⚠️ This is **NOT the actual application UI**
> It is a **showcase website** designed to explain the system, architecture, and capabilities.

## ▶️ How to Run

### Option 1 — Open directly

```
open index.html
```

### Option 2 — Local server (recommended)

```
cd static-site
python -m http.server 8080
```

Then open:

```
http://localhost:8080
```

## 🌍 Deployment

Upload the files to any static hosting:
