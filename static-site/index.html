About the Project

Built for Thoughtful Career Decisions

Career Intelligence Agent is a decision-support system, not a job-application bot. It collects postings from public sources, runs them through a multi-dimensional scoring engine calibrated to your personal profile, and surfaces the evidence you need — without ever touching your data outside your own machine.

The core philosophy: career decisions are high-stakes. Good decisions require grounded evidence, not gut feel or generic keyword matching. Every recommendation the system makes cites the exact skills, projects, or resume fragments that support it.

This is a working system with 600+ passing tests, a full Streamlit dashboard, a local RAG pipeline, and a 7-dimensional scoring engine. Some integrations are partial (Israeli job boards return demo data; email/Slack notifications are configured but disabled by default). What's here is honest about what it does.

🔒

Privacy-First

All data — resume, projects, scores — stays on your machine. Personal profile is gitignored by design.

📊

Evidence-Based

Every recommendation is grounded in your actual documents via local RAG retrieval, not hallucinated.

🧠

Decision Support

You stay in control. The agent advises — it never applies to jobs, sends emails, or acts autonomously.

⚡

No-Dependency Core

TF-IDF indexing, scoring, and filtering work with zero external APIs. LLMs are optional enhancements.

What It Does

Core Capabilities

Real capabilities only — no feature-list fiction.

🎯

7-Dimensional Career Scoring

Every job is scored across seven weighted dimensions — not a single match percentage, but a breakdown you can act on.

  • 25% Skill Overlap
  • 20% Title Relevance
  • 15% Seniority Realism
  • 15% Domain Fit
  • 10% Work Mode Alignment
  • 10% Strategy Alignment
  • 5% Portfolio Match
Apply Now Stretch Not Right Now
📚

Local RAG Pipeline

All retrieval runs on your machine via a pure-Python TF-IDF engine — no OpenAI embeddings required.

  • Loads .md, .txt, .pdf, .json from knowledge_base/
  • Smart paragraph/sentence chunking (~800 chars, 80-char overlap)
  • Inverted TF-IDF index built locally
  • Multi-query retrieval for job analysis
  • Grounded Q&A with source attribution
  • Confidence rating: high / medium / low / none
🔍

Skill Gap Analysis

Identifies exactly what's missing and separates quick wins from hard gaps.

  • Extracts missing skills from job descriptions
  • Categories: easy-to-close vs hard-to-close
  • Flags weak skills from your profile
  • Gap severity: low / medium / high
  • Drives action-plan generation
🗂️

Portfolio Matching

Scores your portfolio projects against each job and recommends what to lead with.

  • Keyword + technology overlap scoring
  • Priority-ordered project list per job
  • Emphasis advice (what to highlight)
  • Integrated with 7-dim scorer (5% weight)
📅

Weekly Strategic Review

Synthesizes a week of job data into actionable strategy — not just a list of scores.

  • Top opportunities ranked by fit
  • Recurring skill gap trends
  • Domain & seniority distribution
  • Focus recommendations: next 7 & 30 days
  • Low-value patterns to stop wasting time on
🌐

Multi-Source Job Collection

Collects from public sources without violating terms of service.

  • RSS: We Work Remotely, RemoteOK
  • ATS: Greenhouse + Lever (public APIs, no auth)
  • Hacker News "Who's Hiring" thread (Algolia API)
  • 4-layer deduplication (URL → ID → fingerprint → fuzzy)
  • Keyword-based filter engine (positive & negative rules)
💬

Career Q&A Interface

Ask questions against your own documents and get grounded, cited answers.

  • "What are my strongest backend skills?"
  • "Which projects match AI/ML roles?"
  • "What's my experience with distributed systems?"
  • Every answer includes the source chunks used
  • No hallucination — confidence rated per answer
🔄

LLM-Optional Architecture

Optional LLM providers enhance synthesis — but nothing breaks without them.

  • Claude (Anthropic) — supported
  • OpenAI GPT-4 — supported
  • Gemini (Google) — supported
  • Ollama (local models) — supported
  • Graceful fallback to rule-based MockLLMProvider

Workflow

How It Works

From raw job posting to actionable recommendation — no cloud required.

01

Configure Your Profile

Edit config/profile.yaml with your target roles, skill set, preferred domains, seniority level, and career tracks. Optionally upload resume and project docs to knowledge_base/. Your personal profile is gitignored — it never leaves your machine.

YAML config + local documents
02

Ingest Your Knowledge Base

Run scripts/ingest_knowledge.py to load, chunk, and index all your career documents. The TF-IDF index is built locally in pure Python — no API calls, no embeddings service. Index persists to data/knowledge_index/.

Local RAG · TF-IDF · Pure Python
03

Collect Jobs

Run scripts/fetch_jobs.py or use the background scheduler. Jobs are pulled from RSS feeds (We Work Remotely, RemoteOK), Greenhouse/Lever ATS public APIs, and Hacker News. 4-layer deduplication prevents re-processing.

RSS · ATS APIs · Hacker News · SQLite
04

Choose Execution Mode

Select how the pipeline runs from the sidebar in the dashboard: Hybrid (RAG + Agent) — career scoring with knowledge base evidence (default). Agent Only — career scoring without KB retrieval. RAG Only — knowledge base retrieval without scoring. Each mode is explicit, deterministic, and shown in the status bar.

Hybrid · Agent Only · RAG Only · Mode status panel
05

Score & Analyze

Each job is filtered by keyword rules, then scored across 7 weighted dimensions using your profile. Simultaneously: gap analysis extracts missing skills, portfolio matching identifies your best-fit projects, action planner generates tailored next steps. In Hybrid mode, local knowledge base evidence is attached to each result.

7-Dim Scoring · Gap Analysis · Portfolio Matching
06

Review in the Dashboard

Open the Streamlit dashboard (streamlit run dashboard/streamlit_app.py). The Decision Console shows scored jobs with recommendation labels ("Apply Now", "Stretch Opportunity", etc.). Ask your career Q&A, review skill trends, and get your weekly strategic summary. The execution status panel shows exactly which mode ran and what components executed.

Streamlit · Decision Console · Career Q&A · Execution Status
06

You Decide

The agent surfaces evidence — you make the call. Mark jobs with feedback signals in the dashboard. The weekly review engine learns from patterns in your scored history. No automatic applications, no emails sent without your action.

Human-in-the-loop · Feedback signals · Weekly strategy

Real-World Application

Use Cases

01

Active Job Search

Run the scheduler to pull new postings every 6 hours. Each morning, check the Decision Console for "Apply Now" recommendations with full evidence: which skills match, which are gaps, which portfolio project to lead with.

Outcome: Apply to fewer jobs, but better-matched ones.
02

Skill Gap Roadmap

Run the weekly review after collecting a batch of roles in your target domain. The trend analysis shows which skills appear most often in gaps — differentiating quick certifications from multi-month learning investments.

Outcome: Prioritized learning plan grounded in real market demand.
03

Salary & Seniority Reality Check

Seniority Realism scoring flags when roles are genuinely above or below your current level. Domain distribution analytics show where the market actually sits vs. where you'd like to be.

Outcome: Calibrated expectations before negotiation.
04

Career Pivot Research

Add a second career track to your profile (e.g., "ML Engineering" alongside "Backend"). The strategy alignment dimension shows which jobs bridge your current experience toward the new direction without requiring a full reset.

Outcome: Identify pivot-friendly roles that compound on current experience.
05

Portfolio Positioning

Before updating your portfolio site, analyze the top 20 jobs you'd actually want. Portfolio Matcher tells you which projects are underemphasized relative to market demand and what to add or highlight.

Outcome: Portfolio tailored to real hiring signals, not guesswork.
06

Company & Domain Research

Feed a specific company's public job board (via Greenhouse/Lever collector) and run the scorer. Get a clear read on what that company values vs. your current profile, before investing in a cover letter or referral.

Outcome: Targeted preparation instead of generic applications.
